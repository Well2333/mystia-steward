# mystia-steward

## 开发环境

- Node.js 22+
- pnpm 10+

## 安装依赖

```bash
pnpm install
```

## 启动开发服务器

```bash
pnpm dev
```

默认地址：<http://localhost:5173>

## 构建

```bash
pnpm build
```

## 说明

- 项目使用 Vite + React + TypeScript。
- 仓库已包含 `pnpm-lock.yaml`，建议统一使用 `pnpm`。
- 首次执行 `pnpm` 时，如果本机启用了 Corepack，可能会自动写入 `packageManager` 字段。
